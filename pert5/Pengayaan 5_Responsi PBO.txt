SOAL PENGAYAAN (Refleksi Polimorfisme)
Jawablah pertanyaan berikut:
1. Apa yang dimaksud dengan polimorfisme dalam OOP? Jelaskan bagaimana metode override bekerja.
2. Mengapa array bertipe DefaultMesin dapat menyimpan objek MesinMotor dan MesinTraktor?
3. Apa perbedaan override dan overload?
4. Apa keuntungan menggunakan polimorfisme dalam sistem inventaris seperti contoh di atas?
5. Bagaimana cara kerja dynamic dispatch pada Java saat memanggil method override?

jawaban
1. polimorfisme adalah kemampuan sebuah objek untuk memiliki banyak bentuk.Method overriding adalah ketika subclass menimpa method dari calss induknya
2. karena mesinMotor dan mesinTraktor adalah subclass dari defaultMesin
3. override dilakukan pada 2 class berbeda (superclass dan subclass), sementara overloading terjadi pada satu class sama. di overriding nama method dan parameter sama, sementara pada overloading nama method sama sementara parameter nya berbeda
4. kode lebih rapih, dinamis, mengurangi duplikasi kode
5. java memilih method berdasarkan tipe objek sebenarnya, bukan tipe variabel
